Venn diagram demo

      venn-beta
        title Very Basic Venn Diagram
        set A
        set B
        set A,B   label: AB
      
      venn-beta
        title Three overlapping sets
        set A
        set B
        set C
        set A,B     label: AB,   background: skyblue
        set B,C     label: BC,   background: orange
        set A,C     label: AC,   background: lightgreen
        set A,B,C   label: ABC,  background: white,      color: red
